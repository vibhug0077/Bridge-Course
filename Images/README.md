# Images

Place supporting images for the course here.

Examples:

- Screenshots of tools.
- Hardware photos.
- Exported chart images.
- Network or architecture illustrations.

## Recommended Image Evidence

For an intensive course submission, this folder can hold:

- Screenshots of C compilation and program output.
- Screenshots of Python notebook execution.
- Excel pivot tables and charts.
- Python and R generated charts.
- System information screenshots for hardware analysis.
- Network command screenshots such as `ping` or IP configuration output.
- Architecture diagrams exported from Mermaid or drawing tools.

## File Quality Guidelines

- Use clear filenames that include the unit and topic.
- Crop screenshots to the relevant area.
- Avoid blurry images.
- Do not include private passwords, tokens, personal identifiers, or sensitive network information.
- Add a short caption or explanation in the Markdown file that references each image.
