# Bridge Course — Networking and Internet Illustrated Package

This folder is a self-contained replacement for:

```text
Unit_1_Computer_Fundamentals/05_Networking_and_Internet.md
```

## Contents

- `05_Networking_and_Internet.md` — detailed theory and practical networking tutorial
- `images/infographics/` — editable SVG diagrams
- `images/infographics/png/` — GitHub-ready PNG diagrams
- `NETWORKING_INFOGRAPHIC_CONTACT_SHEET.jpg` — visual index of all diagrams
- `labs/simple_tcp_server.py` — small classroom TCP server
- `labs/simple_tcp_client.py` — small classroom TCP client
- `MANIFEST.json` — package file list

## GitHub Placement

Copy the Markdown file and the `images` and `labs` folders into:

```text
Bridge-Course/Unit_1_Computer_Fundamentals/
```

The relative image paths in the Markdown file will then render correctly.

## Safety

The networking labs are for devices and networks that the learner owns or is authorised to use. Temporary servers and shares should be stopped after each exercise. Do not disable the firewall globally or enable sharing on public Wi-Fi.
