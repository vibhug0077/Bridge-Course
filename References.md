# References

Use these resources to deepen the topics covered in the bridge course.

## How to Use References

For an intensive bridge course, references should be used actively:

- Read one short section after each lesson.
- Recreate at least one example from the reference.
- Compare the reference explanation with the course note.
- Add unfamiliar terms to a personal glossary.
- Use official documentation when syntax or library behavior matters.

## Computer Fundamentals and Architecture

- Patterson, David A. and Hennessy, John L. *Computer Organization and Design*.
- Stallings, William. *Computer Organization and Architecture*.
- Tanenbaum, Andrew S. *Structured Computer Organization*.
- Bryant, Randal E. and O'Hallaron, David R. *Computer Systems: A Programmer's Perspective*.
- Nisan, Noam and Schocken, Shimon. *The Elements of Computing Systems*.

## C Programming

- Kernighan, Brian W. and Ritchie, Dennis M. *The C Programming Language*.
- Prata, Stephen. *C Primer Plus*.
- cppreference.com for C language reference.
- King, K. N. *C Programming: A Modern Approach*.
- ISO C references and compiler documentation for implementation-specific behavior.

## Python

- Python official documentation: https://docs.python.org/3/
- Sweigart, Al. *Automate the Boring Stuff with Python*.
- VanderPlas, Jake. *A Whirlwind Tour of Python*.
- Downey, Allen B. *Think Python*.
- McKinney, Wes. *Python for Data Analysis*.
- Real Python tutorials for practical examples: https://realpython.com/

## Mathematics and Statistics

## Suggested Reading Path

1. Beginners: course notes, official docs examples, Automate the Boring Stuff, Think Python.
2. C programming depth: C Programming: A Modern Approach, The C Programming Language, cppreference.
3. Systems depth: Computer Systems: A Programmer's Perspective, Computer Organization and Design.
4. Mathematics depth: Discrete Mathematics and Its Applications, OpenIntro Statistics.
5. Visualization depth: Storytelling with Data, The Truthful Art, ggplot2 documentation.
