# Figures

Place exported diagrams, screenshots, or generated figures here.

Most course diagrams are currently embedded directly in Markdown using Mermaid.

## Intensive Figure Set to Create

Students or instructors can export the following diagrams into this folder:

- Complete computer system model.
- Input-process-output-storage cycle for ATM and online exam systems.
- Memory hierarchy diagram.
- Fetch-decode-execute cycle.
- CPU, memory, I/O, and bus architecture diagram.
- LAN to internet web request path.
- Graph theory examples: city graph, prerequisite graph, network graph.
- Data visualization chart selection flow.

## Naming Convention

Use descriptive lowercase filenames with unit numbers:

```text
unit1_ipo_atm.png
unit2_flowchart_largest_three.png
unit4_graph_city_routes.png
unit6_memory_hierarchy.png
```

Every exported figure should have a matching explanation in the relevant lesson, summary, or exercise submission.
