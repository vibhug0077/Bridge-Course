# Illustrated Computer Fundamentals Package

This folder is ready to copy into the repository path:

```text
Bridge-Course/Unit_1_Computer_Fundamentals/
```

## Contents

- `01_Fundamentals_of_Computers.md` — expanded illustrated lesson
- `images/external/` — downloaded open-licence and public-domain images
- `images/infographics/` — editable SVG versions of 24 original English educational infographics
- `images/infographics/png/` — rendered PNG versions used by the lesson
- `INFOGRAPHIC_CONTACT_SHEET.jpg` — quick visual index of every infographic
- `IMAGE_CREDITS.md` — source links, authors and licences

## GitHub Upload

Upload the complete folder without changing its internal structure. The Markdown uses relative paths, so all images render automatically on GitHub.

## Offline Use

The lesson does not require remote image URLs. Download or clone the folder and open the Markdown file in VS Code, Typora, Obsidian, or another Markdown viewer.
